<h1>1- Git </h1>
<h3>Busca da chave</h3>

	*1- Localmente no Git é preciso ter uma chave que serve de identificação do projeto em questão.*

		**Em qualquer lugar do seu computador: Execute, o comando!**
			
			***ssh-Keygen***
			
		**Atenção: O caminho da localização desta seguinte chave esta no mento de sua geração.**
		
	*2-  Em seguida ,entre no diretorio onde a chave foi criada, abra o arquivo  "id_rsa_pub" e copie a chave gerada para adicionar no GitHub.*
		

<h1>2-  GitHub </h1>
<h3>Configuração da sua chave</h3>

	*1- Entre no GITHUB com o seu login e senha*

	*2- Veja o caminho para a inserção da chave:*

		**Configurações (no menu que se encontra no topo);**

		**Configurações (Settings) / Configurações Pessoais (Personal settings)  / SSH Keys**
		
		**Adicione um nome para chave e :**
			
			***finalmente adicione a chave que foi copiada no arquivo do passo anterior.***
			
		**Pronto, o seu ambiente do GITHUB esta funcionando.**
